Madoko uses HTML5 features and requires a recent browser version to work correctly.

Please upgrade your browser to a more recent version.

Madoko is known to work with IE 10+, Chrome 31+, Safari 7+, and Firefox 27+.

Snap! You already have Madoko open in another window.

Unfortunately, Madoko only works within a single browser window.

This page will try to reload automatically after ? seconds...

 

Note: you can work around this limitation by:

  1. Opening Madoko.net in a different browser
  2. Use Chrome incognito mode (Ctrl+Shift+N), or Firefox and IE's private browsing mode (Ctrl+Shift+P), to open another instance of Madoko.net. However, in such mode your local changes are not saved when closing the browser, so make sure your work is synchronized with the cloud. As such, a private mode works best for temporarily viewing other documents.
